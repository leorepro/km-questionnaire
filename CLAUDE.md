# KM Questionnaire

Enterprise Knowledge Management Diagnostic Assessment — a single-page HTML application in Traditional Chinese.

## Project structure

- `km-questionnaire.html` — Self-contained HTML/CSS/JS questionnaire app (Apple-style design)

## Conventions

- Language: Traditional Chinese (zh-TW) for all user-facing text
- Styling: Inline CSS using CSS custom properties (Apple design language)
- No build tools or dependencies — pure vanilla HTML/CSS/JS
